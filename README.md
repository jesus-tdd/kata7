# IS2 - Kata 7
Séptima *kata* de Ingeniería del Software 2 del Grado Ingeniería Informática (ULPGC)


### Ejercicio
> Partiendo de la kata anterior, el objetivo de esta nueva kata es empaquetar y distribuir la aplicación como un producto instalable.
> Para ello, se deberán utilizar las capacidades de Maven para la compilación y empaquetado del proyecto, junto con jpackage,
> la herramienta introducida en Java 14, que permite crear instaladores nativos para diferentes sistemas operativos (Windows, macOS, Linux).
